# Contacts App

This project is the front end of the Contacts app

# Description

This project allows the user to Create, Update, and Delete contacts on the app

# Technologies

* React.js

## Installation
1. Clone the repo
   ```sh
   git clone https://github.com/fbrionesgtz/Contacts.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start server
   ```sh
   npm start
   ```
